# Screenshots

Take these screenshots and save them here:

| Filename | What to capture | Recommended size |
|----------|----------------|-----------------|
| `screensaver.png` | Full-screen screensaver with a game visible (hero image, title, badges, clock) | 1920×1080 or crop to 16:9 |
| `customizer-games.png` | Games tab with game list + detail panel open | ~1400×900 |
| `customizer-display.png` | Display tab with preview visible, elements configured | ~1400×900 |
| `customizer-settings.png` | Settings tab showing enrichment panels | ~1400×900 |
| `tray-menu.png` | System tray icon with right-click menu open | Crop tight |

Tips:
- Use a game with a nice hero image (Cyberpunk, RDR2, etc.)
- Dark theme looks best in screenshots
- Windows: Win+Shift+S for region snip
- Crop out any personal info (Steam ID is fine)
